infinitescroll 参数解释

参数 类型 描述 默认值
loading finished Function 每次加载完成后触发该函数 undefined
finishedMsg String 所有数据加载完成提示文字(它会在间隔speed后消失) 
<em>Congratulations, you've reached the end of the internet.</em>
img String 加载的时候,显示的loading.gif 图片 一张图片地址(data:image/gif...)
msg Object 如何不想使用官方提供的方式，可以在这个进行自定义(需要传入
$("<div>加载中...</div>"))jquery对象
 null
msgText String 加载的时候提示文字(比如："加载中...") 
<em>Loading the next set of posts...</em>
selector String 上面参数msg(显示加载中...)要存放在哪个容器,它的值是一个选择的值 null
speed String | Number msg信息消失的速度,可以"fast"或者是数字100 fast
start Function 每次ajax的时候触发该函数 undefined
state isDuringAjax Boolean false
isInvalidPage Boolean false
isDestroyed Boolean 处在destroyed状态 false
isDone Boolean 加载完毕 false
isPaused Boolean 处在暂停状态 false
isBeyondMaxPage Boolean false
currPage Number 当前页数 false
debug Boolean 启用调试信息，若启用必须引入debug.js false
behavior $(window)
binder Boolean false
navSelector String 分页容器(用来存放参数nextSelector的容器) div.navigation
nextSelector String 下一页链接的选择器 div.navigation a:first
contentSelector String 选择器字符串 null
extraScrollPx Number 滚动条距离底部多少像素的时候开始加载 150
itemSelector div.post
animate Boolean 当有新数据加载进来的时候，页面是否有动画效果 false
animate Boolean 当有新数据加载进来的时候，页面是否有动画效果 false
pathParse Boolean 当有新数据加载进来的时候，页面是否有动画效果 false
dataType String 从服务端返回的数据类型 false
appendCallback Boolean 正常设置为false true
bufferPx Number 提示语展现的时长，数字越大，展现时间越短 40
errorCallback Function ajax请求数据的时候，发生错误就触发该函数 function () { }
infid Number 示例的id，用于区分页面多次用到这个插件 0
pixelsFromNavToBottom undefined
path Function 该函数用来返回url地址(path: function(index) { return "php/infinitescroll.php?page=" + index +"&step=5"; }) undefined
prefill false
maxPage Number 允许ajax最大的页数 undefined
